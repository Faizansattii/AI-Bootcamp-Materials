## ğŸ”§ Troubleshooting Guide

### **Common Issues:**

#### **Issue: "Docker daemon not running"**
**Solution:**
- Windows/Mac: Start Docker Desktop from applications
- Linux: `sudo systemctl start docker`
- Check system tray/menu bar for Docker icon

#### **Issue: "Permission denied" (Linux)**
**Solution:**
```bash
sudo usermod -aG docker $USER
# Log out and log back in
```

#### **Issue: "Port already in use"**
**Solution:**
```bash
# Find what's using port
docker ps
docker stop <container_id>
# Or use different port
docker run -p 8001:8000 ...
```

#### **Issue: "Cannot find Dockerfile"**
**Solution:**
- Check file is named `Dockerfile` (capital D, no extension)
- Check you're in correct directory
- Use `ls` (Linux/Mac) or `dir` (Windows) to list files

#### **Issue: "GitHub Actions not triggering"**
**Solution:**
- Check file is in `.github/workflows/` (note the dot!)
- Check YAML syntax (indentation matters)
- Enable Actions: Settings â†’ Actions â†’ Allow all actions
